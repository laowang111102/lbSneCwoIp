# Python学生就业管理系统 python1069

## 项目概述

本项目是一款基于Python的学生就业管理系统，旨在帮助学院、企业和学生实现就业信息的有效管理。

## 技术栈

- 前端：HTML、CSS、JavaScript、jQuery
- 后端：Python、Django
- 数据库：MySQL
- 开发工具：PyCharm、Navicat

## 系统角色

- 学院管理员
- 专业管理员
- 企业管理员
- 学生

## 运行环境

- Python 3.0以上
- MySQL 8.0

## 功能模块

### 二级功能模块

- 学院信息管理
- 专业信息管理
- 企业信息管理
- 岗位信息管理
- 管理员信息管理
- 学生信息管理

### 三级功能模块

- 学生教育经历
- 学生项目经历
- 学生投递记录
- 修改个人信息
- 密码修改
